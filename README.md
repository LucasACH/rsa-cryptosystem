### Prerequisites

* [sympy](https://pypi.org/project/sympy/) 
* [progress](https://pypi.org/project/progress/)

### Installation

1. Install dependencies
 ``` pip install sympy progress ```
  
3. Clone the repo
   ```sh
   git clone https://github.com/LucasACH/rsa-cryptosystem.git
   ```
